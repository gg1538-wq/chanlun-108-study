# 缠论108课学习 PWA — 手机使用说明

## 方式一：局域网内部分享（推荐，最私密）

适合：手机与电脑连**同一个 WiFi**，只给自己或家人/同事用，完全不暴露到互联网。

1. 双击 `启动局域网分享.cmd`
2. 窗口会显示 `http://192.168.x.x:7100` 地址
3. 同一 WiFi 下的 iPhone/iPad/其他电脑在浏览器打开即可；iPhone 可「添加到主屏幕」
4. 关闭 cmd 窗口即停止分享

注意：局域网 http 下 iOS 不提供离线缓存（功能在线时完整）；首次启动若弹防火墙提示，勾选"专用网络"并允许。

## 方式二：互联网临时隧道（异地也能开）

适合：手机不在同一 WiFi 时的临时分享。

## 快速开始（iPhone，方式二）

1. 在电脑上双击 `启动手机访问.cmd`
2. 等黑色窗口里出现 `https://xxxxxxxx.lhr.life` 地址和二维码
3. 用 iPhone 相机扫码（或 Safari 输入该网址）
4. 打开后点 Safari 底部分享按钮 →「添加到主屏幕」
5. 之后从主屏幕图标打开，就是全屏 App 体验；断网也能用（首次需完整打开一遍让离线缓存生效）

## 注意事项

- **电脑需开机**：隧道是从你手机连到这台电脑的通道，关闭 cmd 窗口或电脑关机后手机即无法访问
- **网址每次会变**：免费隧道每次启动生成新地址，需要重新扫码打开
- **学习数据跟网址走**：你的进度、复盘记录保存在手机本机浏览器里，绑定当时的网址；换新网址后数据不互通。如需迁移，先在旧地址里用「首页 → 导出我的数据」，再在新地址里「导入」
- **功能不受影响**：108 课全文、154 张图、搜索、词典、复盘在每次会话内全部可用

## 长期使用建议

临时隧道适合体验和验收。若要固定网址、电脑不开机也能用，有两个方向：

1. **localhost.run 固定域名（免费）**：注册 localhost.run 账号并绑定 SSH 公钥，可获得长期不变的专属域名
2. **静态托管（最省心）**：GitHub Pages / Cloudflare Pages / Netlify / Vercel 任一免费静态托管，部署后 7×24 在线，与电脑无关
